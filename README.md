# fff: fast file finder

fff is a simple file finder that prints the path to a desired file in the terminal

## Installation

#### AUR

```sh
yay -S fff-dev
```
#### Git

```sh
git clone https://github.com/rgcastrof/fff.git && cd fff && sudo make install
```

## Usage

```sh
fff -d DIR -f FILE
```


## Uninstallation

```sh
cd fff
sudo make uninstall
```
